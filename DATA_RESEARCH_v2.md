## 🧠 심리학 자료 수집

### 1. 나이대별 연애 심리 특성

#### 10대 (15-19세)
**심리학적 기반**: 정체성 형성기 (Erikson), 첫사랑 효과

| 특성 | 설명 | 질문지 반영 |
|------|------|------------|
| 호ormonal 변화 | 감정 기복 심함 | 감정 중심 질문 |
| 집단 의식 | 친구 의견 중요 | 친구 관련 질문 |
| 순수함 | 조건 덜 따짐 | 단순한 호감 질문 |
| 극단적 | 흑백 논리 | 명확한 선택지 |

**실제 사례**:
> "10대 때는 그냥 좋으면 좋은 거예요. 친구들이 응원해주면 더 설레고, 카톡 답장 빨리 오는 게 최고예요." 
> - 19세 여성, P 성향

#### 20대초반 (20-24세)
**심리학적 기반**: 친밀감 대 고립 갈등 (Erikson), 자율성 추구

| 특성 | 설명 | 질문지 반영 |
|------|------|------------|
| 탐색 욕구 | 다양한 경험 원함 | 유연한 선택지 |
| 외모 중시 | 시각적 매력 중요 | 외모 관련 질문 |
| SNS 중심 | 디지털 소통 | 연락 수단 질문 |
| 자유로움 | 결혼 압력 없음 | 가벼운 관계 질문 |

**실제 사례**:
> "대학생 때는 썸 타는 게 재밌어요. 과CC도 해보고 동아리 선배도 좋아해봤어요. 딱히 미래를 안 보고 그냥 재밌는 대로."
> - 23세 남성, D 성향

#### 20대중반 (25-29세)
**심리학적 기반**: 직업 정체성 확립, 현실적 고려 시작

| 특성 | 설명 | 질문지 반영 |
|------|------|------------|
| 현실 검토 | 조건 따지기 시작 | 현실적 질문 |
| 장기 관계 | 안정성 추구 | 미래 관련 질문 |
| 경제 고려 | 금전/직업 중요 | 조건 관련 질문 |
| 시간 압박 | 30대 다가옴 | 서두르는 심리 |

**실제 사례**:
> "26살부터는 진지하게 만나요. 직업도 안정돼야 하고, 상대도 성실한 사람 찾게 돼요. 그냥 호감만으로는 부족해요."
> - 27세 여성, R 성향

#### 20대후반 (28-29세)
**심리학적 기반**: 결혼 적령기 압박, 선택적 친밀감

| 특성 | 설명 | 질문지 반영 |
|------|------|------------|
| 결혼 의식 | 배우자 조건 중시 | 결혼 관련 질문 |
| 효율 중시 | 시간 낭비 싫음 | 빠른 판단 질문 |
| 안정 추구 | 변화 두려움 | 안정성 질문 |
| 가족 고려 | 부모님 의견 중요 | 가족 관련 질문 |

**실제 사례**:> "29살이면 진짜 마지막이라는 생각이 들어요. 이 사람이랑 결혼할 수 있을까? 진짜 심각하게 따져봐요."
> - 29세 남성, C 성향

#### 30대 (30세 이상)
**심리학적 기반**: 생성감 대 침체감, 성취 중심

| 특성 | 설명 | 질문지 반영 |
|------|------|------------|
| 현실 최우선 | 조건 매우 중요 | 구체적 조건 질문 |
| 시간 효율 | 빠른 결정 선호 | 단기 판단 질문 |
| 경험 기반 | 실패 경험 반영 | 학습 관련 질문 |
| 가족 중심 | 자녀/가정 고려 | 미래 가정 질문 |

**실제 사례**:
> "30대에는 진짜 현실이에요. 연봉, 집, 가치관 다 맞아야 해요. 감정만으로는 못 만나요."
> - 32세 여성, R 성향

### 2. 성별차이 연애 심리

#### 남성 연애 특성

| 영역 | 특성 | 설명 |
|------|------|------|
| 시각 중심 | 외모 우선 | 1차 매력 중요 |
| 문제 해결 | 해결책 중심 | 감정 위로보다 해결 |
| 공간 독립 | 혼자 시간 필요 | 밀착보다 적절한 거리 |
| 직선적 | 명확한 신호 선호 | 애매함보다 확실함 |

**성향별 남성 특징**:
- **남성+D**: 적극적 추진, 빠른 고백, 경쟁 의식
- **남성+C**: 관찰 후 접근, 확신 있을 때 움직임
- **남성+R**: 조건 비교, 현실적 판단, 효율 중시
- **남성+P**: 드물지만 깊은 감정, 표현은 서툼

#### 여성 연애 특성

| 영역 | 특성 | 설명 |
|------|------|------|
| 감정 공유 | 대화 중심 | 감정 나누기 중요 |
| 안전 추구 | 확실한 신호 필요 | 신뢰 구축 우선 |
| 세심함 | 작은 변화 감지 | 디테일 중요 |
| 미래 지향 | 장기 관계 고려 | 미래 상상 중요 |

**성향별 여성 특징**:
- **여성+D**: 적극적 표현, 원하는 건 직접 말함
- **여성+C**: 깊은 관찰, 확실한 신호 받은 후 움직임
- **여성+R**: 현실 조건 따짐, 합리적 관계 추구
- **여성+P**: 감정 중심, 분위기와 설렘 중요

### 3. 조합별 심리 프로필

#### P1: 20대초반 + 여성 + D (직진파)
**프로필**: 대학생, 적극적, 설렘 추구
**질문지 특성**: 
- 직접적, 활동적 질문
- 친구/SNS 관련 질문 포함
- 빠른 결과 원함

#### P1: 20대초반 + 남성 + D (직진파)
**프로필**: 대학생, 적극적, 경쟁적
**질문지 특성**:
- 외모/첫인상 중시 질문
- 바로 행동으로 옮기는 질문
- 단순 명확한 선택지

#### P1: 20대초반 + 여성 + C (신중파)
**프로필**: 대학생, 분석적, 깊은 관계 추구
**질문지 특성**:
- 시간/패턴 관련 질문
- 친구 관찰 관련 질문
- 확실한 신호 중시

#### P1: 20대중반 + 여성 + R (합리파)
**프로필**: 사회초년생, 현실적, 안정 추구
**질문지 특성**:
- 직업/경제 조건 관련
- 미래 계획 관련
- 효율성 관련 질문

#### P1: 20대중반 + 남성 + D (직진파)
**프로필**: 사회초년생, 적극적, 성과 중심
**질문지 특성**:
- 빠른 진행 선호 질문
- 성과/반응 중시 질문
- 직접적 접근 관련

## 📝 질문지 설계 프레임워크

### 테스트 1: 썸이냐? (Sum Test)

#### 공통 베이스 (모든 조합)
```javascript
const sumCommonBase = [
  { type: 'contact', text: '일주일 연락 빈도는?', options: [...] },
  { type: 'meeting', text: '상대가 먼저 만나자고 한 적?', options: [...] },
  { type: 'atmosphere', text: '함께 있을 때 분위기는?', options: [...] },
  { type: 'response', text: '내 연락에 대한 반응은?', options: [...] },
  { type: 'future', text: '미래 약속(다음 만남)은?', options: [...] }
];
```

#### 나이별 추가 질문
- **10대**: "친구들이 우리 사이 어떻게 생각해요?"
- **20대초반**: "SNS에서 서로 태그하거나 스토리 봐주나요?"
- **20대중반**: "서로의 직업/미래에 관심을 보이나요?"
- **20대후반**: "결혼이나 진지한 관계에 대한 언급이 있나요?"
- **30대**: "실질적인 조건(집, 경제)에 대한 대화가 있나요?"

#### 성별 추가 질문
- **남성**: "상대의 외모/스타일에 대해 칭찬한 적이 있나요?"
- **여성**: "상대가 내 감정을 잘 배려해주나요?"

#### 성향별 추가 질문
- **D**: "직접 만나자고 제안한 후 상대 반응은?"
- **C**: "시간이 지날수록 관계가 깊어지는 느낌인가요?"
- **R**: "이 관계의 실질적인 이점이 있나요?"
- **P**: "함께 있을 때 감정적으로 따뜻함을 느끼나요?"
