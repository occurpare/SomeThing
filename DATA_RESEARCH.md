# SomeThing 데이터 수집 및 분석 보고서

## 1. 심리학 이론 기반 연애 성향 분석

### 1.1 성향 4가지 정의 (D/C/R/P)

#### 🔥 D (Direct/직진파) - 행동 중심형
**심리학적 기반**: 행동주의적 접근, 즉각적 보상 추구

| 특성 | 설명 |
|------|------|
| 의사결정 | 직관적, 빠른 판단 |
| 갈등 대처 | 즉각적 해결, 숨기지 않음 |
| 연애 시작 | 적극적 표현, 빠른 진전 |
| 장점 | 솔직함, 에너지 넘침 |
| 단점 | 상대 부담, 무리한 접근 |

**실제 사례**:
> "좋아하는 사람 생기면 일주일 안에 고백해요. 애매한 시간 보내는 거 싫어해요. 한 번 좋아하면 바로 보채고 선물도 주고 데이트 신청도 해요."
> - 26세 여성, D 성향

---

#### 🤔 C (Cautious/신중파) - 분석 중심형
**심리학적 기반**: 인지심리학, 위험 회피 성향

| 특성 | 설명 |
|------|------|
| 의사결정 | 정보 수집 후 판단 |
| 갈등 대처 | 신중한 관찰, 시간 투자 |
| 연애 시작 | 확인 후 진행, 확실한 신호 필요 |
| 장점 | 배려심, 깊은 관계 |
| 단점 | 기회 놓침, 지나친 신중 |

**실제 사례**:
> "상대방이 나한테 관심 있는지 한 달은 지켜봐요. 카톡 답장 속도, 만날 눈빛, 친구들 언급 여부까지 다 체크해요."
> - 28세 남성, C 성향

---

#### 📊 R (Rational/합리파) - 논리 중심형
**심리학적 기반**: 합리적 선택 이론, 비용-수익 분석

| 특성 | 설명 |
|------|------|
| 의사결정 | 논리적 분석, 조건 비교 |
| 갈등 대처 | 효율적 해결책 모색 |
| 연애 시작 | 적합성 검토 후 진행 |
| 장점 | 현실감, 안정성 |
| 단점 | 냉정해 보임, 감정 소홀 |

**실제 사례**:
> "연애할 때 장래, 경제력, 가치관 다 따져봐요. 감정만으로는 결정 안 해요. 로망보다는 실리를 봐요."
> - 30세 여성, R 성향

---

#### 💭 P (Passionate/감성파) - 감정 중심형
**심리학적 기반**: 감정심리학, 알렉산더 커뮤니케이션 이론

| 특성 | 설명 |
|------|------|
| 의사결정 | 감정에 따른 직관 |
| 갈등 대처 | 감정 표현, 공감 중시 |
| 연애 시작 | 분위기, 감정적 연결 |
| 장점 | 따뜻함, 진정성 |
| 단점 | 감정 기복, 충동적 |

**실제 사례**:
> "분위기 딱 좋을 때 호감 느껴요. 노래 나오고 날씨 좋고 상대 눈빛이 따뜻하면 바로 빠져요."
> - 24세 남성, P 성향

---

### 1.2 연애 성향 테스트 문항 설계 (10문)

#### 문항 1: 좋아하는 사람이 생겼을 때
- **D**: 바로 표현하고 적극적으로 다가간다
- **C**: 상대의 반응을 보며 신중하게 접근한다
- **R**: 먼저 조건과 상황을 따져본다
- **P**: 감정이 이끄는 대로 분위기를 타본다

#### 문항 2: 연애에서 나의 스타일
- **D**: 솔직하게 감정을 표현하는 편
- **C**: 차근차근 신뢰를 쌓아가는 편
- **R**: 효율과 실리를 중시하는 편
- **P**: 감정과 설렘을 중시하는 편

#### 문항 3: 애매한 관계에서
- **D**: 확실한 답을 원해 바로 묻는다
- **C**: 시간을 두고 상대 마음을 살핀다
- **R**: 관계의 실익을 따진다
- **P**: 감정의 흐름을 느끼며 기다린다

#### 문항 4: 연애 상대 선택 기준
- **D**: 호감이 가는 대로
- **C**: 검증된 신뢰감
- **R**: 현실적인 조건
- **P**: 감정적 호흡

#### 문항 5: 이성에게 호감 느낄 때
- **D**: 적극적으로 리드
- **C**: 내 마음부터 정리
- **R**: 객관적 분석
- **P**: 분위기 따라가기

#### 문항 6: 연애 중 갈등
- **D**: 즉각 해결
- **C**: 시간 두고 관찰
- **R**: 논리적 해결책
- **P**: 감정적 대화

#### 문항 7: 이상형 생각할 때
- **D**: 명확한 스타일
- **C**: 검증된 성격
- **R**: 조건 리스트
- **P**: 느낌과 분위기

#### 문항 8: 새로운 사람에게 끌림
- **D**: 바로 다가간다
- **C**: 신중한 관찰
- **R**: 조건 확인
- **P**: 감정적 반응

#### 문항 9: 연락 기다릴 때
- **D**: 먼저 연락
- **C**: 기다리며 분석
- **R**: 다른 일 하며 유효시간 계산
- **P**: 감정적으로 설레거나 불안

#### 문항 10: 이별 후
- **D**: 빨리 다음 사랑
- **C**: 깊은 회복 시간
- **R**: 교훈 도출 후 다음
- **P**: 감정적으로 힘들어

## 2. 테스트별 맞춤 질문지

### 2.1 테스트 1: 썸이냐? (Sum-Pass/No)

**공통 문항 (8개)**:
1. 연락 빈도 확인
2. 함께 있는 시간 즐거운지
3. 상대방 시선 방향
4. 신체적 거리감
5. 친구들 반응
6. 서로 미래 언급 여부
7. 연락 답장 속도
8. 만남 이후 연락

**직진파(D)용 추가 (4개)**:
- 직접 물어본 적 있는지
- 내 호감 표현 정도
- 상대방 반응의 적극성
- 다음 만남 확정 여부

**신중파(C)용 추가 (4개)**:
- 분위기 변화 감지
- 작은 제스처 관찰
- 친구들 사이 평판
- 패턴 분석 결과

**합리파(R)용 추가 (4개)**:
- 시간 대비 호감도
- 조건 일치 여부
- 효율성 계산
- 객관적 상황 판단

**감성파(P)용 추가 (4개)**:
- 함께 있을 때 감정
- 눈빛의 따뜻함
- 분위기 호흡감
- 감정적 연결감

---

### 2.2 테스트 2: 나.. 얘 좋아하냐? (Do I Like?)

**공통 문항 (6개)**:
1. 자주 생각나는 정도
2. 연락 왔을 때 기분
3. 다른 이성과 비교
4. 미래 상상 여부
5. 질투 여부
6. 고백 하고 싶은 충동

**직진파(D)용 추가 (4개)**:
- 행동으로 옮긴 적
- 적극성 수준
- 결과 무관 대응
- 에너지 투여량

**신중파(C)용 추가 (4개)**:
- 확인된 신호
- 검증된 정보
- 시간 투자 대비 감정
- 확신 정도

**합리파(R)용 추가 (4개)**:
- 실익 분석
- 조건 부합도
- 대안 비교
- 효율성 판단

**감성파(P)용 추가 (4개)**:
- 감정 기복
- 분위기에 따른 변화
- 직감적 확신
- 마음의 설렘

---

### 2.3 테스트 3: 얘.. 나 좋아하냐? (Does She/He Like Me?)

**공통 문항 (6개)**:
1. 먼저 연락 여부
2. 나를 볼 때 표정
3. 내 이야기 기억력
4. 친구들에게 나 언급
5. 신뢰하고 털어놓는 정도
6. 나를 위하는 행동

**직진파(D)용 추가 (4개)**:
- 직접적인 표현 여부
- 적극적 접근 빈도
- 확실한 호감 신호
- 밀고 당김 패턴

**신중파(C)용 추가 (4개)**:
- 패턴 분석 결과
- 일관성 있는 관심
- 시간 흐름에 따른 변화
- 확정적 신호

**합리파(R)용 추가 (4개)**:
- 객관적 투자 정도
- 시간 대비 호감 비율
- 조건적 접근 여부
- 계산된 행동

**감성파(P)용 추가 (4개)**:
- 따뜻한 감정 느낌
- 눈빛에 담긴 정
- 분위기 타는 정도
- 감정적 접근

## 3. 결과 해석 기준

### 3.1 점수 계산법

**연애 성향 테스트 (10문)**:
- 각 선택에 성향별 가중치 (1~3점)
- 총점 가장 높은 성향 선택
- 동점 시: 가장 최근 선택 성향 우선

**썸 판별 테스트**:
- 문항별 0~10점
- 총점 → % 환산
- 80%+: 썸 진행중
- 60~79%: 썸 가능성
- 40~59%: 애매
- 40% 미만: 친구 수준

**내/상대 마음 테스트**:
- 문항별 0~10점
- 점수별 해석 제공
- 감정 강도 판단

### 3.2 성향별 결과 화면 구성

**직진파 결과**:
> "당신은 직진파! 핵심 키워드: 적극적, 솔직함, 빠른 추진"
> -> 빠른 실전 조언 제공

**신중파 결과**:
> "당신은 신중파! 핵심 키워드: 분석, 관찰, 깊은 관계"
> -> 신중한 접근법 제안

**합리파 결과**:
> "당신은 합리파! 핵심 키워드: 현실, 효율, 논리"
> -> 실익 중심 조언

**감성파 결과**:
> "당신은 감성파! 핵심 키워드: 감정, 분위기, 진정성"
> -> 감정적 교류법 제안
